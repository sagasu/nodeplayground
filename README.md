# nodeplayground
Playground for node

### App using mongo
```
npm install
npm install -g karma
npm install -g karma-cli
```
http://localhost:3000/


### Smaller app
```
node ./js/expressApp.js
```
http://localhost:8888/


### Run app
```
npm start
```

### Run tests app
```
npm test
```

### Run code in languageFeatures folder
```
npm run moduleScope
```
